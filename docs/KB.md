# 团队知识库

> **维护方式**：运行 `/sync-context kb` 从 knowledge-base MCP 自动同步，或手动添加。
> **最后同步**：未初始化

## 如何初始化

1. 复制 `.claude/settings.json.example` 为 `.claude/settings.json`，配置 knowledge-base MCP
2. 运行 `/sync-context kb`，Claude 会将相关知识条目追加到本文件
3. 或直接在下方手动添加团队规范和架构决策

## 内容说明

- 团队编码规范、约定俗成的做法
- 关键架构决策（ADR）
- 常见问题与解决方案
- 外部系统对接注意事项
